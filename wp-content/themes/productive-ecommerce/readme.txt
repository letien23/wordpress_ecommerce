A free e-Commerce theme for WordPress

=== Description ===
Productive eCommerce theme for WordPress is an has been designed ground up to easily adapt to almost any online store.

Features:
- Translation ready (sample translation included)
- Support for multiple layouts
- Support for specific layout for specific pages/posts
- Easy to install
- Easy to integrate with popular plugins
- Easy to adapt and extend
